# Shunya Marketplace

Welcome to **Shunya Marketplace**, a sleek and stylish web application I built to streamline ordering and tracking for local food enthusiasts! As an aspiring Artificial Intelligence & Robotics Engineer, I wanted to step outside my comfort zone and build a clean, responsive web interface that showcases my ability to develop full-stack applications and integrate dynamic features.

## What is Shunya?
Shunya is a digital marketplace that allows users to quickly browse a curated menu of food items, completely customize them to their taste, and place an order using a dynamic cart system. 
More than just an ordering page, it features an integrated tracking system so customers can easily monitor the real-time status of their orders.

## Key Features
- **Dynamic Cart System**: Add to cart, adjust quantities, and see a live-updating price breakdown - all seamlessly handled via JavaScript.
- **Customizable Orders**: A beautifully designed modal lets you adjust modifiers for your selected items (like "double egg" or "veggies++") before adding them to your cart.
- **Real-Time Database Tracking**: Powered by Firebase Realtime Database, which allows customers to create a tracking ID and password on checkout and monitor whether their order is Pending, Confirmed, or Delivered.
- **Vendor Dashboard**: A dedicated vendor view for live monitoring of incoming orders and changing their status on the fly.
- **Modern Aesthetics**: Built without heavy styling frameworks—just pure, vanilla CSS to maintain absolute control over the dynamic animations, responsive layouts, and glass-morphic premium look.
- **Post-Delivery Feedback**: Once an order is marked as delivered, customers can rate their experience using an interactive 5-star rating system directly from their order page.

## Tech Stack
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ESModules)
- **Backend & Database**: Firebase Realtime Database
- **Icons**: Phosphor Icons
- **Fonts**: Google Fonts (Outfit)
